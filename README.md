# 🏠 House Price Prediction System

A Machine Learning mini project that predicts house prices using Linear Regression.

## 🚀 Features
- Flask Web Application
- CLI Version
- Machine Learning Prediction
- Responsive UI
- Ngrok Support

## 🧠 Machine Learning Used
- Linear Regression
- Supervised Learning

## ⚙️ Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- Bootstrap
- Ngrok

## ▶️ Run Web Version

```bash
python app.py
```

Open:
```bash
http://127.0.0.1:5000
```

## ▶️ Run CLI Version

```bash
python app_cli.py
```
